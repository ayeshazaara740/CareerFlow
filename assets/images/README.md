- `office-scene.svg` — office / workplace backdrop

- `testimonial-1.svg`, `testimonial-2.svg` — candidate portraits
- `saved-jobs.svg`, `empty-saved.svg` — empty state illustrations
- `logo.svg` — CareerFlow logo
